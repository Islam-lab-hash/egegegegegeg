# ДЛЯ ПОКУПАТЕЛЯ

---

## КАК ЗАКАЗАТЬ ТОВАР

### Шаг 1: Откройте каталог

Нажмите кнопку «Каталог» в главном меню.

Вы увидите 11 категорий:

- Халаты
- Пеньюары
- Сорочки
- Комплекты
- Наборы трусиков
- Свадебное белье
- Корсетные изделия
- Чулки и колготки
- Аксессуары
- Распродажа
- Новинки

### Шаг 2: Выберите категорию

Нажмите на интересующую категорию.

Откроется список товаров с фото, названием и ценой.

### Шаг 3: Откройте карточку товара

Нажмите на товар.

Вы увидите:

- Фотографии товара
- Название
- Цену
- Описание
- Доступные размеры
- Цвета
- Материал
- Страну производства

### Шаг 4: Добавьте в корзину

Нажмите «В корзину».

Если нужно выбрать размер: нажмите «Уточнить размер» и напишите свой размер.

### Шаг 5: Перейдите в корзину

Нажмите «Корзина» в главном меню.

Вы увидите:

- Список добавленных товаров
- Количество каждого товара
- Цены
- Общую сумму

Кнопки в корзине:

- «Оформить заказ» - перейти к оформлению
- «Очистить корзину» - удалить все товары
- «← В меню» - вернуться в главное меню

### Шаг 6: Оформите заказ

Нажмите «Оформить заказ».

Введите данные по шагам:

**Шаг 1: Имя**
Напишите ваше имя.

**Шаг 2: Телефон**
Напишите ваш номер телефона.

После этого бот покажет реквизиты для оплаты:

- Номер карты
- Имя получателя
- Банк
- Сумма к оплате

Вы можете:

- Скопировать реквизиты
- Оплатить переводом
- Прислать чек менеджеру

После оплаты нажмите «✅ Я оплатил».

Заказ создастся со статусом «Ожидает подтверждения».

## ПОДБОР ТОВАРА

### Как работает подбор

Нажмите «Подбор» в главном меню.

Вы увидите баннер и варианты подбора:

- По размеру
- По цвету
- По цене
- По материалу
- Подарок

### По размеру

Нажмите «По размеру».

Выберите ваш размер:

- XS (42-44)
- S (44-46)
- M (46-48)
- L (48-50)
- XL (50-52)
- XXL (52-54)

Бот покажет все товары вашего размера.

### По цвету

Нажмите «По цвету».

Выберите цвет:

- Белый
- Чёрный
- Красный
- Розовый
- Бордовый
- Синий
- Бежевый

Бот покажет все товары выбранного цвета.

### По цене

Нажмите «По цене».

Выберите ценовой диапазон:

- До 2000 ₽
- 2000 - 5000 ₽
- 5000 - 10000 ₽
- От 10000 ₽

Бот покажет товары в вашем бюджете.

### По материалу

Нажмите «По материалу».

Выберите материал:

- Шёлк
- Хлопок
- Атлас
- Кружево
- Микрофибра
- Вискоза

Бот покажет товары из выбранного материала.

### Подарок

Нажмите «Подарок».

Выберите повод:

- День рождения
- Годовщина
- 8 марта
- День всех влюблённых
- Другое

Бот покажет подходящие подарки.

## ЧАСТЫЕ ВОПРОСЫ

**Как изменить количество товара в корзине?**
Сейчас нельзя. Удалите товар и добавьте заново.

Откройте корзину, нажмите на товар, нажмите «Удалить из корзины».

## УПРАВЛЕНИЕ ТОВАРАМИ

### Добавить товар

 1. Нажмите «📦 Товары»
 2. Нажмите «➕ Добавить товар»
 3. Выберите категорию
 4. Введите название товара
 5. Введите цену
 6. Введите размеры через запятую (например: S, M, L, XL)
 7. Введите цвета через запятую (например: чёрный, белый)
 8. Введите описание (можно пропустить)
 9. Введите материал (можно пропустить)
10. Введите страну производства (можно пропустить)
11. Прикрепите фото товара

Товар добавлен.

### Редактировать товар

1. Нажмите «📦 Товары»
2. Нажмите «✏️ Редактировать товар»
3. Выберите категорию
4. Выберите товар
5. Нажмите на параметр, который хотите изменить:
   - Название
   - Цену
   - Размеры
   - Цвета
   - Описание
   - Материал
   - Страну производства
   - Фото
6. Введите новое значение

Товар обновлён.

### Удалить товар

1. Нажмите «📦 Товары»
2. Нажмите «🗑 Удалить товар»
3. Выберите категорию
4. Выберите товар
5. Подтвердите удаление

Товар удалён.

## УПРАВЛЕНИЕ ЗАКАЗАМИ

### Посмотреть все заказы

1. Нажмите «🛒 Заказы»
2. Выберите фильтр:
   - Все заказы
   - Ожидают оплаты
   - Оплачены
   - В обработке
   - Завершены
   - Отменены

Вы увидите список заказов с датой, именем, суммой и статусом.

### Посмотреть детали заказа

Нажмите на заказ.

Вы увидите:

- Номер заказа
- Имя покупателя
- Телефон
- Способ получения
- Адрес
- Список товаров
- Сумму
- Статус
- Дату создания

### Подтвердить оплату

1. Откройте заказ
2. Нажмите «✅ Подтвердить оплату»
3. Выберите способ доставки:
   - 🏪 Самовывоз из магазина
   - 📦 Доставка через Ozon
   - 🚗 Курьер по городу
4. Если курьер или Ozon: введите адрес/пункт выдачи

Заказ перейдёт в статус «Оплачен».

Покупатель получит уведомление с деталями доставки.

### Отклонить заказ

1. Откройте заказ
2. Нажмите «❌ Отклонить»
3. Введите причину (опционально)

Заказ перейдёт в статус «Отменён».

Покупатель получит уведомление.

## УПРАВЛЕНИЕ АДМИНИСТРАТОРАМИ

⚠️ Только для суперадмина!

### Как посмотреть список админов

1. Нажмите «👥 Администраторы»
2. Вы увидите список всех админов с их ID и никнеймами

### Как добавить админа

1. Попросите пользователя написать боту /myid
2. Пользователь пришлёт вам свой ID
3. Нажмите «👥 Администраторы»
4. Нажмите «➕ Добавить админа»
5. Введите ID пользователя
6. Подтвердите

Новый администратор получит доступ к админ-панели.

### Как удалить админа

1. Нажмите «👥 Администраторы»
2. Нажмите «🗑 Удалить админа»
3. Выберите админа из списка
4. Подтвердите

Администратор потеряет доступ к админ-панели.
