# MyStoreBot — Telegram-бот для магазина белья

Минималистичный бот-магазин: каталог, карточки товаров с фото, корзина, оформление заказа, подбор, информация о магазине и **админ-панель** для управления каталогом.

## 📁 Структура

```
MyStoreBot/
├── bot.py              # точка входа
├── config.py           # тексты + чтение .env
├── database.py         # SQLite (категории, товары, корзины, заказы, юзеры)
├── states.py           # FSM-состояния
├── keyboards.py        # все инлайн-клавиатуры
├── handlers_user.py    # хендлеры покупателя
├── handlers_admin.py   # админ-панель
├── requirements.txt
├── .env                # ваши токены (НЕ коммитьте!)
└── store.db            # создаётся автоматически при первом запуске
```

## 🚀 Локальный запуск на вашем ПК

### 1. Установите Python 3.10+
Проверьте версию:
```bash
python --version
```
(на Windows может быть `python`, на macOS/Linux — `python3`)

### 2. Установите зависимости
В терминале откройте папку проекта и выполните:
```bash
pip install -r requirements.txt
```

### 3. Проверьте `.env`
Файл `.env` уже создан с вашими данными:
```
BOT_TOKEN=8871876182:AAGfCu3CjH5tx0qFKZ7d4QLrAyjxlxLIgsw
ADMIN_IDS=7582523959
```
Если админов несколько — перечислите ID через запятую.

### 4. Запустите
```bash
python bot.py
```
В терминале должно появиться `Бот @your_bot запущен`. Откройте Telegram → найдите вашего бота → `/start`.

Чтобы остановить — `Ctrl+C`.

## 🛠 Админ-панель

Команда `/admin` доступна только пользователям из `ADMIN_IDS`. Возможности:

- ➕ **Добавить товар** — пошаговый мастер (категория → название → цена → размеры → цвета → материал → страна → описание → фото)
- 🗑 **Удалить товар** — выбор категории → клик по товару
- 📁 **Категории** — список, добавление, удаление (удаление категории удалит и её товары)
- 📊 **Статистика** — пользователи, товары, заказы
- 📣 **Рассылка** — массовое сообщение всем, кто запустил бота

## 🛒 Что умеет бот для покупателя

- **Каталог** с категориями (11 предзаполнено: Комплекты, Халаты, Пеньюары, Сорочки, Наборы трусиков, Свадебное бельё, Корсетные изделия, Чулки и колготки, Купальники, Аксессуары, Акции)
- **Карточка товара** — фото, название, цена, размеры, цвета, материал, страна, описание, кнопки: В корзину / Уточнить размер / Спросить консультанта
- **Подбор** — по размеру, по цвету, для подарка, помощь консультанта
- **Корзина** — добавление/удаление, общая сумма, очистка
- **Оформление заказа** — имя, телефон (через контакт или вручную), город, способ получения (Самовывоз/Доставка/Ozon), комментарий. Заявка автоматически уходит всем админам.
- **Информация** — адреса, график, доставка, оплата, обмен/возврат, Instagram, оператор

## ✏️ Где поменять тексты магазина

Откройте `config.py` и отредактируйте:
- `SHOP_NAME` — название
- `SHOP_ADDRESSES` — адреса
- `SHOP_SCHEDULE` — график
- `DELIVERY_INFO`, `PAYMENT_INFO`, `EXCHANGE_INFO`
- `INSTAGRAM_URL` — ссылка на инстаграм
- `OPERATOR_USERNAME` — username менеджера (с @)

## 📦 Деплой на хост

Когда захотите перенести на сервер — подойдёт любой VPS (Timeweb, Beget, Hetzner, Aeza и т.п.). Алгоритм:
1. Заливаете папку на сервер (через `git`, `scp` или панель).
2. `pip install -r requirements.txt`.
3. Запуск через `systemd` или `pm2`/`screen`/`tmux`.
4. БД `store.db` лежит рядом — можно бэкапить файл целиком.

## 📞 WhatsApp-бот

WhatsApp Business API требует отдельной интеграции (Twilio, 360dialog, Meta Cloud API) — это уже **платный** канал. Когда будете готовы — сделаем зеркало этого функционала через WhatsApp Cloud API (бесплатно до 1000 диалогов/мес).
