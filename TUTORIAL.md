# ПОЛНЫЙ ТУТОРИАЛ: БОТ-МАГАЗИН @indira_official_bot

## ЧАСТЬ 1: ПОКУПАТЕЛЬ

### Главное меню

При старте бот показывает 4 кнопки:

1. **КАТАЛОГ** - все товары по категориям
2. **ПОДБОР** - подобрать белье по параметрам
3. **КОРЗИНА** - ваши выбранные товары
4. **ИНФО** - информация о магазине

---

### 1. КАТАЛОГ

**Как пользоваться:**

1. Нажмите "КАТАЛОГ"
2. Появится фото-баннер и список категорий:
   - Комплекты
   - Халаты
   - Пеньюары
   - Сорочки
   - Наборы трусиков
   - Свадебное белье
   - Корсетные изделия
   - Чулки и колготки
   - Купальники
   - Аксессуары
   - Акции

3. Выберите категорию (например, "Комплекты")
4. Появится список товаров в категории
5. Нажмите на товар для подробной информации

**Карточка товара показывает:**
- Фото товара
- Название
- Цену
- Размеры
- Цвета
- Материал
- Страна производства

**Действия в карточке товара:**
- "В корзину" - добавить товар в корзину
- "Уточнить размер" - задать вопрос о размере
- "Спросить консультанта" - связаться с оператором

---

### 2. ПОДБОР

**Помощь в выборе белья по параметрам.**

Нажмите "ПОДБОР" и выберите:

1. **По размеру**
   - Введите ваш размер
   - Бот покажет подходящие товары

2. **По цвету**
   - Белый
   - Черный
   - Графит
   - Небесно-голубой

3. **По цене**
   - До 2000 руб
   - 2000-5000 руб
   - От 5000 руб

4. **По материалу**
   - Шелк 100%
   - Хлопок
   - Микрофибра

5. **Подарок**
   - Подборка для подарка
   - Красивая упаковка

6. **Оператор**
   - Связь с живым консультантом

---

### 3. КОРЗИНА

**Как работает корзина:**

1. Нажмите "КОРЗИНА"
2. Если корзина пуста: появится сообщение "Корзина пуста"
3. Если есть товары:
   - Фото-баннер корзины
   - Список всех товаров с ценами
   - Общая сумма
   - Реквизиты для оплаты

**Действия в корзине:**
- "Оформить заказ" - начать оформление
- "Очистить корзину" - удалить все товары
- "В меню" - вернуться в главное меню

---

### 4. ОФОРМЛЕНИЕ ЗАКАЗА

**Пошаговый процесс:**

**Шаг 1: Имя**
- Введите ваше имя
- Пример: "Анна"

**Шаг 2: Телефон**
- Введите номер телефона
- Пример: "+7 999 123-45-67"

**Шаг 3: Подтверждение**
- Проверьте введенные данные
- Нажмите "Все верно" или "Изменить"

**Что происходит дальше:**
- Заказ создается со статусом "Ожидает оплаты"
- Вам показываются реквизиты для оплаты
- Администратор получает уведомление о новом заказе

**Реквизиты оплаты:**
- Номер карты
- Имя получателя
- Название банка

После оплаты:
- Отправьте чек менеджеру для подтверждения
- Администратор подтвердит оплату
- Выберете способ получения заказа

---

### 5. СПОСОБЫ ПОЛУЧЕНИЯ

После подтверждения оплаты администратором:

**Варианты доставки:**
1. **Самовывоз**
   - Бесплатно
   - Из магазина по адресу

2. **Курьер**
   - По городу
   - Укажите адрес доставки

3. **Ozon**
   - Доставка через пункты выдачи Ozon
   - Выберите удобный пункт

---

### 6. ИНФОРМАЦИЯ О МАГАЗИНЕ

Нажмите "ИНФО" для разделов:

**Адреса магазинов**
- Список всех точек

**График работы**
- Часы работы
- Выходные дни

**Доставка**
- Самовывоз: бесплатно
- Курьер: по тарифам
- Ozon: по тарифам Ozon

**Оплата**
- банковская карта
- Наличные при получении

**Обмен/возврат**
- Условия обмена
- Сроки возврата

**Instagram**
- Ссылка на Instagram магазина

**Связь с оператором**
- Прямая связь с менеджером

## ЧАСТЬ 2: АДМИНИСТРАТОР

**Как войти:**
1. Откройте бот
2. Напишите команду `/admin`
3. Если вы администратор: откроется панель управления

**Кто может войти:**
- Суперадмин (ваш ID: 7582523959)
- Все добавленные администраторы

---

### ГЛАВНОЕ МЕНЮ АДМИНКИ

```
ТОВАРЫ       КАТЕГОРИИ
ЗАКАЗЫ       ТЕКСТЫ МАГАЗИНА
РЕКВИЗИТЫ    СТАТИСТИКА
```

---

### 1. УПРАВЛЕНИЕ ТОВАРАМИ

**Для входа:**
- Нажмите "ТОВАРЫ"

**Что можно делать:**

#### Добавить товар
1. Нажмите "Добавить товар"
2. Выберите категорию
3. Введите название
4. Введите цену
5. Введите размеры (через запятую)
6. Введите цвета (через запятую)
7. Введите материал
8. Введите страну производства
9. Прикрепите фото или нажмите "Пропустить"
10. Товар добавлен

#### Просмотреть товары
1. Нажмите "Список товаров"
2. Выберите категорию
3. Появится список всех товаров
4. Нажмите на товар для редактирования

#### Редактировать товар
В карточке товара нажмите нужную кнопку:
- "Название" - изменить название
- "Цена" - изменить цену
- "Размеры" - изменить размеры
- "Цвета" - изменить цвета
- "Материал" - изменить материал
- "Страна" - изменить страну производства
- "Фото" - заменить фото
- "Удалить" - удалить товар

#### Удалить товар
1. Откройте товар
2. Нажмите "Удалить"
3. Подтвердите удаление

---

### 2. УПРАВЛЕНИЕ КАТЕГОРИЯМИ

**Для входа:**
- Нажмите "КАТЕГОРИИ"

**Что можно делать:**

#### Добавить категорию
1. Нажмите "Добавить категорию"
2. Введите название
3. Категория создана

#### Редактировать категорию
1. Нажмите "Редактировать"
2. Выберите категорию
3. Измените название
4. Сохраните

#### Удалить категорию
1. Нажмите "Удалить"
2. Выберите категорию
3. Подтвердите удаление
4. Все товары в этой категории удалятся автоматически

---

### 3. УПРАВЛЕНИЕ ЗАКАЗАМИ

**Для входа:**
- Нажмите "ЗАКАЗЫ"

**Статусы заказов:**
- `awaiting_payment` - ожидает оплаты
- `paid` - оплачен, ожидает доставки
- `ready` - готов к выдаче
- `delivered` - доставлен
- `rejected` - отклонен

**Что можно делать:**

#### Просмотреть все заказы
1. Нажмите "Все заказы"
2. Появится список последних заказов

#### Просмотреть заказ
1. Нажмите на ID заказа
2. Появится подробная информация:
   - ID заказа
   - Покупатель (имя, телефон)
   - Товары
   - Сумма
   - Статус
   - Способ доставки
   - Дата создания

#### Подтвердить оплату
1. Откройте заказ со статусом "awaiting_payment"
2. Нажмите "Оплата подтверждена"
3. Бот предложит покупателю выбрать способ доставки
4. Статус сменится на "paid"

#### Отклонить заказ
1. Откройте заказ
2. Нажмите "Отклонить"
3. Статус сменится на "rejected"

#### Изменить статус
Доступные переходы:
- awaiting_payment → paid, rejected
- paid → ready, delivered
- ready → delivered

## ЧАСТЬ 4: ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Бот не отвечает

**Причины:**
1. Бот выключен
2. Ошибка в коде
3. Проблема с токеном

**Решение:**
1. Проверьте статус службы на странице Hosting → Services
2. Посмотрите логи ошибки в `/dev/shm/mystorebot_err.log`

### Конфликт с другим ботом

**Ошибка:**
"Conflict: terminated by other getUpdates request"

**Решение:**
- Убедитесь, что бот запущен только в одном месте
- Если запускали на ПК - выключите

### Товары не показываются

**Причины:**
- База данных пуста
- Нет товаров в категории

**Решение:**
1. Проверьте через админку наличие товаров
2. Добавьте товары вручную

## ЧАСТЬ 6: ТЕХНИЧЕСКИЕ ДЕТАЛИ

**Где хранятся данные:**
- База данных: `/home/workspace/MyStoreBot/store.db`
- Фото товаров: `/home/workspace/MyStoreBot/assets/products/`
- Баннеры разделов: `/home/workspace/MyStoreBot/assets/`

**База данных содержит таблицы:**
- `categories` - категории товаров
- `products` - товары
- `cart` - корзина
- `orders` - заказы
- `users` - пользователи
- `admins` - администраторы
- `settings` - настройки и тексты

**Как изменить фото-баннеры:**
1. Замените файлы в `/home/workspace/MyStoreBot/assets/`:
   - `catalog.png` - баннер каталога
   - `selector.png` - баннер подбора
   - `cart.png` - баннер корзины
   - `info.png` - баннер информации
2. Перезапускать бот не нужно, фото обновятся автоматически
